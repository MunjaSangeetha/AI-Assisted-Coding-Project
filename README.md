# ☕ Sitara Coffee

A demo coffee shop website built with HTML, CSS, and JavaScript.

**Live site:** https://munjasangeetha.github.io/sitara-coffee/

## Features
- Home page with hero section
- Menu with category filters (Hot, Cold, Food)
- Cart drawer with quantity controls
- Order page (UI demo — no real payments)
- Login / Signup (UI demo — no real authentication)
- Thank You confirmation page

## Folder Structure
```
sitara-coffee/
├── index.html        ← main page
├── css/
│   └── style.css     ← all styles
├── js/
│   ├── menu-data.js  ← menu items data
│   └── app.js        ← cart & page logic
└── README.md
```

## Note
This is a **UI prototype only**. No real orders are placed, no payments are processed, and no personal data is stored or transmitted.
